# SendReactMessage Client

### Send React Message

``rtk.sendReactMessage`` - Is a function that allows you to send a message to a React interface.

- type: `function`
- parameters: 
    - `value`: `table`
        - `action`: `string`
        - `data?`: `any`
    - `options?` : `table`
        - `focus?`: `boolean` | `{ [1]: boolean, [2]: boolean }` - Focus
        - `locations?`: `string` | `vec2` | `{ [x]: number, [y]: number }` | `{ number, number }` - Location
            - `string` as `top-left`, `top-right`, `bottom-left`, `bottom-right` or `center`
        - `keepInput?`: `boolean`  - Keep input
- return: `void`

```lua
rtk.sendReactMessage({
    action = "example",
    data = { message = "Hello, World!" }
}, {
    focus = true,
    locations = 'center',
    keepInput = true
})
```
